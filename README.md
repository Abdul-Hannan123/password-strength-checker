# 🔐 Password Strength Checker

A simple and responsive Password Strength Checker built using **HTML, CSS, and JavaScript**.
It gives real-time feedback to users about the strength of their password.

---

## 🚀 Features

* Detects password strength in real-time
* Shows:

  * ❌ Weak Password
  * ⚠️ Medium Password
  * ✅ Strong Password
* Input border changes color based on strength
* Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 💡 How It Works

* When the user types a password:

  * If it's short or simple → **Weak**
  * If it has some complexity → **Medium**
  * If it includes uppercase, lowercase, numbers, and symbols → **Strong**
* JavaScript handles validation and updates UI dynamically
